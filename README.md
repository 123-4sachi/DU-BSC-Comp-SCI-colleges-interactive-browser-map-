## Delhi NCR Colleges & Metro Interactive Map

### What This Does

This map shows 20 Delhi University colleges and 30+ metro stations on real street maps. Click any marker to see details. Zoom in and out. Pan around. That's it.

### Why I Made This

Finding colleges and their nearest metro stations was hard. This map solves that problem. Now you see everything at once.

### Features

- Real interactive map using OpenStreetMap
- Blue markers for colleges
- Red markers for metro stations
- Each metro line has its own color
- Toggle colleges or metros on/off
- Click markers for information

### How to Use

Save the file. Open in browser. Explore.

### Technical Details

Built with HTML, CSS, JavaScript, and Leaflet.js. Leaflet handles the mapping. OpenStreetMap provides the tiles. No database needed. One file does everything.

### What You Learn

Which colleges are where. Which metro line goes there. The actual distance between places. Simple.
